About Subscription Licenses
Monthly Validation

Subscription licenses require monthly server validation. The check occurs on the same day of the month as activation.

Grace Period

If the server is unreachable, the license remains valid for the grace period (default 7 days).

Cancellation

When canceled, the license becomes invalid on the next check. If reactivated, the browser will detect it on next validation.
